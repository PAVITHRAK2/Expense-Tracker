# 💰 Expense Tracker

## Overview

Expense Tracker is a desktop application developed using **Core Java** and **Java Swing** to help users manage their personal finances efficiently. The application allows users to record daily expenses, maintain monthly salary information, monitor spending patterns, and generate monthly reports through an interactive graphical interface.

The project is designed using a layered architecture consisting of **Model, Repository, Service, and UI** layers, making the application modular, maintainable, and easy to extend.

---

## Features

- Add new expenses
- View all recorded expenses
- Update existing expenses
- Delete expenses
- Manage monthly salary
- Dashboard displaying total salary, total expenses, and savings
- Monthly expense reports
- Spending insights and category-wise analysis
- Persistent file-based data storage
- Interactive desktop GUI built with Java Swing

## Project Architecture

The application follows a layered architecture to separate responsibilities:

- **Model Layer** – Represents the application's data objects.
- **Repository Layer** – Handles reading and writing data to files.
- **Service Layer** – Implements business logic such as expense management, report generation, and spending analysis.
- **UI Layer** – Provides the graphical user interface using Java Swing.
